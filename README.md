## 소개

인간관계까지 케어해주는 경조사 관리 서비스, 경비서의 웹 서비스입니다.

## 로컬 개발 환경

경비서를 로컬에서 개발하려면, .env.example에 포함된 모든 환경 변수를 설정해야 합니다.

모든 환경 변수를 설정했다면 다음 커맨드를 입력해 로컬 환경에서 앱을 실행할 수 있습니다.

```
npm install
npm run dev
```
